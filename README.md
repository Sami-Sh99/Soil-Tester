# Soil-Tester
Soil Tester that predicts the best fit plant for the soil quality


Connect the arduino to the Raspberry pi via USB for Serial communication.
Arduino connections:
Sensor		Arduino
Humidity    	7
Moisture	    A3
LDR		        A4
pH meter    	A5
IR reciever1	A0
IR reciever2	A1
IR reciever3	A2
